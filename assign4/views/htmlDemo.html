Audio Gallery



The Audio Gallery section contains HTML 5 audio players. The audio controls tag is used to specify that the audio player should include controls for playing and pausing the audio. Within audio tag, the source tag specifies the source of the audio file, in this case a .mp3 file with a source URL. The type attribute specifies the type of audio file being used. The audio player will be able to play the audio file if the browser supports .mp3 audio files.

Video Gallery



The Video Gallery section contains HTML 5 video players. TML 5 video player with a width of 400 pixels. The "controls" attribute enables playback controls such as play, pause, and volume. The "source" element specifies the video source and the type of video file (mp4 or ogg). The text between the "video" tags will display in case the browser does not support HTML5 video.
